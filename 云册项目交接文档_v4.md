## 一、项目概述

**尚唯云册**是一个家庭相册管理系统，照片源位于 `G:\照片`，通过 Web 服务提供 PC/移动端浏览和手机端上传功能。

### 核心特性
- **双导航栏UI**：效仿"尚唯云色"设计——第一栏彩色徽章切换大类，第二栏下拉菜单切换细分
- **9大分类**：生活/寸照/房产/卡/头像/医疗/游戏/证件/啄木鸟
- **标签系统**：14种事件标签 + 人物标签 + 年份筛选
- **照片上传**：手机端独立上传页，自动按命名规则建文件夹
- **回忆模式**：全屏轮播 + 背景模糊
- **照片查看器**：缩放/旋转/键盘导航

## 三、技术栈

## 五、v4.0 核心设计（当前版本）

### 5.1 双导航栏架构

完全效仿 **尚唯云色**（`G:\AI\GL`）的设计风格：

**第一导航栏（品牌徽章行）** — 7个彩色徽章：

| 徽章 | 配色 | 映射分类 | 功能 |
|------|------|----------|------|
| 照片 | 金色 #d4a853 | LS | 浏览全部照片，按归属/事件/年份筛选 |
| 生活 | 红色 #ff4444 | YL | 生活相关相册 |
| 回忆 | 橙色 #ff8c00 | ZMN | 全屏回忆轮播 |
| 人物 | 紫色 #9b59b6 | TX/ZJ | 头像和证件相关 |
| 地点 | 粉色 #ff6b9d | - | 按地点筛选（待扩展） |
| 文档 | 青色 #00d4aa | CZ/FC/KA | 证件/房产/卡 |
| 工具 | 蓝色 #4d9fff | YX | 上传/扫描/统计 |

**第二导航栏（内容导航）** — 左侧编码戳 + 右侧链接 + 搜索框：
- 支持 hover 下拉菜单（效仿云色的 `.dropdown-content` 模式）
- 下拉内容包括：归属(Seth/Roger/S&R)、事件标签(14种)、年份(2000-2026)

### 5.2 关键CSS类名


### 5.3 照片数据结构

```javascript
// 相册对象
{
  id: "LS-002",           // 分类代码-序号
  name: "相册名称",
  path: "相对路径",       // 相对于 G:/照片/LS/ 的路径
  date: "2024-01-15",     // 日期
  count: 12,              // 照片数量
  cover: "封面文件名.jpg",
  hasVideo: false,
  tags: ["旅游", "游玩"],
  photos: [
    { f: "文件名.tif", w: 5196, h: 7086, s: 2885881, d: "" }
    // f=文件名, w=宽, h=高, s=大小, d=日期
  ]
}
```

### 5.4 照片路径构建

```javascript
// 缩略图路径
'/thumb/' + catCode + '/' + album.path + '/' + photo.f + '?w=400'

// 原图路径
'/media/G/%E7%85%A7%E7%89%87/' + catCode + '/' + album.path + '/' + encodeURIComponent(photo.f)
```

### 5.5 数据加载模式

**重要：DataLoader 使用纯回调模式（兼容Android 4.4.2），不支持Promise。**

```javascript
// 正确用法
DataLoader.load('LS', function(albums) {
  // albums 是相册数组
});

// 错误用法（会导致崩溃）
DataLoader.load('LS').then(...)  // ❌ DataLoader没有then方法
```

## 六、后端API路由

| 路由 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页（index.html） |
| `/media/G/...` | GET | 媒体代理（G:/... → 支持Range请求） |
| `/thumb/...` | GET | 缩略图（自动生成+缓存） |
| `/api/upload` | POST | 照片上传（multipart/form-data） |
| `/api/scan` | POST | 触发增量扫描 |

### 上传功能

上传时服务器会：
1. 按 `[归属]-[日期]-[事件]-[人物/地点]` 规则创建文件夹
2. 保存照片到 `G:\照片\{文件夹名}\`
3. 自动触发 `auto_scan.py` 更新网站数据

## 十、开发注意事项

### 10.1 PowerShell 环境

- 本系统使用 PowerShell，**不支持 `&&` 连接命令**，必须用 `;` 分号
- 错误写法：`cd dir && python script.py`
- 正确写法：`Set-Location "dir"; python script.py`

### 10.2 文件编辑限制

- **Write 工具**只能在 `G:\AI\MV` 目录内写入文件
- 编辑 PZ 目录文件需要：先在 MV 创建 → 再用脚本复制到 PZ
- 参考 `deploy_v4.py` 的部署模式

### 10.3 ES5 兼容性

- 所有前端 JavaScript 必须使用 ES5 语法（兼容 Android 4.4.2）
- 不使用：`let`/`const`、箭头函数 `=>`、`Promise`、`async`/`await`、模板字符串 `` ` ``
- 使用：`var`、`function(){}`、回调模式

### 10.4 编码注意

- server.py 对路径做 URL 重写：`G:/照片/...` → `/media/G/照片/...`
- 文件名可能含中文和特殊字符，需要用 `encodePathSegments()` 逐段编码
- config.js 提供了 `toMediaPath()` 和 `toThumbPath()` 工具函数

### 10.5 参考项目

- **尚唯云色**（`G:\AI\GL`）：设计风格参考，双导航栏设计的原型
- 关键参考文件：`G:\AI\GL\index.html`（导航栏HTML结构）、`G:\AI\GL\engine\style.css`（导航栏CSS）
