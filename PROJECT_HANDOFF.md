## 一、项目概述

**尚唯云册**是一个家庭相册管理系统，支持 PC/移动端 Web 访问，以及 TV 端展示。

### 核心特性
- 🎨 **黑色主题**：默认深色背景，彩色标签系统
- 🔍 **智能搜索**：按日期、人物、事件、地点多维度搜索
- 📅 **时间轴展示**：按年份浏览照片（1987-2025）
- 🔒 **安全控制**：局域网/广域网权限分离，私密内容保护
- ⚡ **性能优化**：懒加载、缓存清理、图片压缩

## 五、运行方式


## 八、文件命名规范

照片文件命名格式：

```
[归属]-[日期]-[事件]-[描述]-[地点].jpg
```

**示例**：
- `R-19940503-游玩-20岁生日-沈阳北陵公园.jpg`
- `S&R-19990403-结婚典礼-沈阳湘珍水鱼城.jpg`
- `S-19940102-聚会-105高中同学-沈阳松园酒家.jpg`

**归属编码**：
- `R` = Roger（老公）
- `S` = Seth（老婆）
- `S&R` = 夫妻共同

## 十、注意事项

1. **端口**：默认 8084，可在启动时指定
2. **权限**：私密内容（证件、医疗等）仅局域网可访问
3. **缓存**：缩略图缓存在 `cache/` 目录，定期自动清理
4. **备份**：定期备份 `data/` 目录下的索引文件
