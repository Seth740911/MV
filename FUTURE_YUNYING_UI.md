## 🎨 设计特点

### 1. 顶部导航栏
- **品牌Logo**：图标 + 文字
- **圆角搜索框**：带焦点发光效果（紫色光晕）
- **操作按钮**：刷新/设置（圆形按钮）

### 2. 多标签页系统
- **6个标签**：首页、电影、电视剧、动画片、纪录片、收藏
- **Tab图标**：SVG矢量图标
- **激活效果**：底部紫色指示条 + 背景高亮
- **流畅动画**：淡入效果 + 卡片悬浮

### 3. 卡片设计
- **悬浮效果**：上移 6px + 阴影 + 紫色边框发光
- **海报比例**：2:3（标准电影海报）
- **播放按钮**：中心悬浮紫色圆形播放图标
- **评分标签**：右上角金色评分

### 4. 首页布局
- **精选推荐**：横向滚动卡片（大尺寸）
- **最新电影**：网格布局
- **热门剧集**：网格布局

### 5. 移动端适配
- **底部导航**：4个快捷Tab（首页/电影/剧集/收藏）
- **响应式网格**：自适应屏幕宽度

## 📝 适配说明

当前UI是为**影视库**设计的，如果用于**云册照片项目**，需要调整：

### 标签页调整
```
原设计：首页 | 电影 | 电视剧 | 动画片 | 纪录片 | 收藏
云册版：首页 | 相册 | 时间轴 | 人物 | 地点 | 收藏
```

### 卡片调整
```
原设计：电影海报（2:3 竖版）
云册版：相册照片（1:1 或 4:3）
```

### 功能调整
```
原设计：视频播放、演员信息、评分
云册版：照片浏览、EXIF信息、标签筛选
```
