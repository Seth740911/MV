## 🌐 网络协议支持

### 双栈配置 (IPv4 + IPv6)
所有 Python HTTP 服务均启用**双栈监听**:
- **IPv4**: `0.0.0.0` (监听所有IPv4接口)
- **IPv6**: `::` (监听所有IPv6接口, 通过 `IPV6_V6ONLY=0` 同时接受IPv4)

- **本机IPv4**: `192.168.0.10`
- **子网掩码**: `255.255.255.0`

## 🌍 广域网访问实现

### 1. Cloudflare Tunnel 方案
**核心组件**:
- **cloudflared**: Cloudflare官方隧道工具
- **Gateway (8080)**: 路由分发代理
- **tunnel_url.txt**: 动态隧道URL存储

**访问流程**:
```
用户浏览器
    ↓ HTTPS
Cloudflare边缘网络
    ↓ 加密隧道
cloudflared (本地)
    ↓ HTTP
Gateway (127.0.0.1:8080)
    ↓ 路由分发
后端服务 (8081-8086, 8888)
```

**配置要点**:
```python
# Gateway 监听本地端口
server = http.server.HTTPServer(('127.0.0.1', 8080), GatewayHandler)

# 路由规则: 路径前缀 → 端口
SERVICES = {
    '/gl/': 8081,   # 云色
    '/mv/': 8082,   # 云影
    '/yy/': 8083,   # 云音
    '/pz/': 8084,   # 云册
    '/qy/': 8085,   # 云听
    '/fifa/': 8086, # FIFA
    '/apk/': 8888,  # APK下载
}
```

### 2. DNS TXT记录方案
**动态URL发现**:
- **域名**: `tunnel.sethshi.dynv6.net`
- **记录类型**: TXT
- **内容**: `https://xxx.trycloudflare.com`
- **更新频率**: 每30秒检查一次

**自动更新流程**:
```python
# TunnelPublisher 线程
while True:
    if url_changed:
```

## 🔧 服务实现细节

### 1. 端口监听 (所有服务)
**代码位置**: `server.py` (第1240-1247行)
```python
class DualStackTCPServer(socketserver.ThreadingTCPServer):
    address_family = socket.AF_INET6
    def server_bind(self):
        self.socket.setsockopt(socket.IPPROTO_IPV6, socket.IPV6_V6ONLY, 0)
        super().server_bind()

server = DualStackTCPServer(('::', listen_port), MediaServer)
```

**关键点**:
- `AF_INET6`: 使用IPv6协议族
- `IPV6_V6ONLY=0`: 允许IPv6 socket同时接受IPv4连接
- `('::', port)`: 绑定到所有网络接口

### 2. Gateway路由分发
**代码位置**: `gateway.py` (第66-183行)

**路由优先级**:
1. **路径前缀匹配**: `/gl/`, `/mv/` 等
2. **Referer头匹配**: 从Referer中提取服务前缀
3. **默认路由**: 8086 (FIFA)

**请求转发**:
```python
target_url = f'http://localhost:{target_port}{stripped_path}'
headers['Host'] = f'localhost:{target_port}'
headers['X-Forwarded-Proto'] = 'https'
headers['X-Forwarded-Host'] = orig_host
```

### 3. HTML响应处理
**Base标签注入** (解决相对路径问题):
```python
if target_port != DEFAULT_PORT and 'text/html' in content_type:
    base_tag = f'<base href="{prefix}">'
    html = html.replace('<head>', f'<head>{base_tag}', 1)
```

## 🛡️ 安全与访问控制

### 1. 局域网策略
- **强制IP访问**: 所有服务必须通过 `192.168.0.10` 访问
- **禁用域名**: 禁止使用 `sethshi.dynv6.net` 等域名
- **原因**: 确保离线可用性、降低延迟

### 2. 广域网安全
- **Cloudflare防护**: DDoS攻击防护、速率限制
- **HTTPS加密**: 所有广域网流量加密传输
- **隧道隔离**: 后端服务不直接暴露到公网

### 3. 防火墙配置
- **入站规则**: 允许 8081-8086, 8888 端口
- **出站规则**: 允许 cloudflared 连接 Cloudflare网络

## 📊 网络流量统计

### 1. 局域网流量
- **直接访问**: 无代理开销,低延迟
- **带宽**: 受限于千兆以太网 (1Gbps)
- **并发**: 无限制 (受服务器性能约束)

### 2. 广域网流量
- **Cloudflare隧道**: 免费计划支持无限制带宽
- **延迟**: 增加约50-200ms (取决于地理位置)
- **压缩**: 自动gzip压缩
